# Enera Template

Systaliko UI enera website template.

## Stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **Shadcn UI**
- **Systaliko UI**
- **Motion** (animations)
- **Lenis** (smooth scroll)
- **Embla Carousel**
- **Base UI** / **shadcn**
- **Vercel Analytics**
